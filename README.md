# Coders_of_Delhi
# 🚀 CodeBook User Data Analysis (Pure Python)

## 📖 Introduction

Congratulations! 🎉

You have just been hired as a **Data Scientist Intern at CodeBook – The Social Media for Coders**. This Delhi-based company is offering you a **₹10 LPA package** if you successfully complete your **1-month internship**.

But before you get there, you must prove your skills using **only Python** — **no Pandas, no NumPy, and no fancy libraries!**

Your manager, **Puneet Kumar**, has assigned you your first task:

> Analyze a data dump of CodeBook users using pure Python and uncover insights about the platform's users and their interactions.

Your mission is to load, clean, and explore the data to understand its structure and extract meaningful information.

---

## 🎯 Project Objective

This project focuses on performing **data analysis using core Python concepts only**, including:

* Lists
* Dictionaries
* Sets
* Tuples
* Loops and Conditional Statements
* Functions
* File Handling
* JSON Processing

The goal is to understand how real-world data can be analyzed without relying on external data science libraries.

---

## 📊 Tasks Performed

* Load raw user data from JSON files
* Explore the structure of the dataset
* Analyze user information and relationships
* Count users and categories
* Find patterns and statistics within the data
* Build logic for social network recommendations
* Understand how social graph data can be represented using Python data structures

---

## 🛠️ Technologies Used

* Python 3
* JSON
* Jupyter Notebook
* Core Python Data Structures

**No external libraries such as Pandas, NumPy, or Scikit-Learn were used.**
